# cmlib

Computational Modeling Library

Python Packages to support the analysis of different model of computation.

- Finite State Automata
- Regular Expressions
- Linear Temporal Logic
- Discrete-Time Markov Chains
- Synchronous Dataflow
- Max-Plus Algebra
